# NPM

### New Project Setup###

1. Check version

   ```
   npm -v
   ```

2. Install nodemon

   ```
   npm install -g nodemon
   ```

3. Initialize npm to create package.json file

   ```
   npm init
   ```

4. Install packages/save as dependencies in package.json file

   ```
   npm install --save pad //simple tool that left and right pads strings
   ```
